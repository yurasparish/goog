Базовые свойства шрифта в CSS

В следующей таблице показаны базовые свойства шрифта в CSS, позволяющие изменять внешний вид текста в документах.

Свойства Цель
font-family Указывает используемый шрифт.
font-size Указывает используемого шрифта.
font-style Указывает, должен ли шрифт быть обычным, курсивным или наклонным.
font-weight Указывает, должен ли шрифт быть обычным, жирным, жирнее или тоньше.
